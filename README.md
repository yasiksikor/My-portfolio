## Проекты
### 📊 1. Исследовательский анализ бизнес-показателей маркетплейса 
- **Описание:** обзор ключевых метрик и метрик монетизации маркетплейса, поиск инсайтов, подведение результатов эксперимента (предоставление бонуса за первую покупку среди пользователей канала). 
- **Результат:** проект подсветил проблему убытков от пользователей конкретного канала привлечения и отток пользователей в разрезе покупательского сегмента. Эксперимент успешен, но требует поэтапного внедрения с мониторингом долгосрочных эффектов unit-экономики. 
- **Используемый стек:** Python, Pandas, Matplotlib, Seaborn, Scipy.

### 🔍 2.Ad hoc аналитика онлайн-библиотеки в ClickHouse
- **Описание:** создание выделенного аналитического слоя на ClickHouse для оперативного ответа на бизнес-вопросы продакт-менеджера. Задачи включали анализ географической популярности продукта, популярность авторов и их произведений, сравнение метрик между платформами и оценку корректности тегов контента.
- **Результат:** выявила популярные города для деятельности продукта, выделила топ произведений по взаимодействию с пользователями, обнаружены расхождения в тегировании для 1% книжного каталога.
- **Используемый стек:** ClickHouse
- 
  🔗 https://github.com/yasiksikor/ad_hoc_ClickHouse
